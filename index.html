🔒 Política de Privacidade

📋 Sobre este documento

Esta Política de Privacidade descreve como o aplicativo NGB AutoRoad coleta, usa e protege as informações no seu dispositivo Android. Ao usar o app, você concorda com os termos aqui descritos.

Última atualização: Junho de 2026  |  Versão: 1.6.x

🎯 Finalidade do aplicativo

O NGB AutoRoad é um assistente para motoristas de aplicativo (Uber, 99, inDrive, DiDi, Cabify) que analisa corridas oferecidas e exibe um card flutuante com pontuação e informações relevantes para ajudar na tomada de decisão.

♿ Uso do Serviço de Acessibilidade

O NGB AutoRoad usa o Serviço de Acessibilidade do Android exclusivamente para:

  • Ler as telas dos apps de transporte (Uber, 99, inDrive, DiDi, Cabify)
  • Extrair dados de corridas oferecidas: valor, distância, duração, avaliação do passageiro
  • Calcular uma pontuação e exibir um card flutuante com as informações

O app NÃO usa o Serviço de Acessibilidade para:

  • Ler mensagens, e-mails, contatos ou fotos
  • Capturar senhas ou dados bancários
  • Monitorar outros aplicativos além dos de transporte listados
  • Transmitir dados para servidores externos

📊 Dados coletados e armazenados

O NGB AutoRoad armazena localmente no seu dispositivo os seguintes dados para funcionamento:

  • Dados de corridas: valor, distância, duração, pontuação, app de origem — usados para histórico e estatísticas locais
  • Configurações do usuário: preferências de layout, filtros e parâmetros de pontuação
  • Estatísticas de uso: número de corridas aceitas/recusadas — armazenadas apenas localmente

✅ Todos os dados ficam exclusivamente no seu dispositivo. Nenhuma informação é enviada para servidores externos, nuvem ou terceiros.

🌐 Dados NÃO coletados

O NGB AutoRoad não coleta:

  • Nome, CPF, e-mail ou qualquer dado de identificação pessoal
  • Localização GPS ou histórico de rotas
  • Dados financeiros, bancários ou de pagamento
  • Contatos, fotos, arquivos ou mensagens
  • Dados de outros aplicativos além dos de transporte

🔒 Permissões utilizadas

  • Serviço de Acessibilidade: leitura de telas dos apps de transporte (ver seção acima)
  • Exibir sobre outros apps (Overlay): exibir o card flutuante sobre o app de transporte
  • Internet: verificação de atualizações do app (apenas versão, sem dados pessoais)
  • Foreground Service: manter o monitoramento ativo em segundo plano
  • Vibrar: alertas opcionais de corridas boas/ruins

🔄 Atualizações do aplicativo

O app pode verificar atualizações consultando o GitHub Releases. Nessa consulta, nenhum dado pessoal é enviado — apenas uma requisição anônima para verificar a versão mais recente disponível.

👶 Crianças

O NGB AutoRoad é destinado exclusivamente a motoristas adultos de aplicativo. Não coletamos intencionalmente dados de menores de 18 anos.

📝 Alterações nesta política

Esta política pode ser atualizada com novas versões do app. Recomendamos verificar periodicamente. A data de "última atualização" no topo indica a versão mais recente.
